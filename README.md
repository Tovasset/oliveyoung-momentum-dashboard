# 올리브영 · 아마존 K-뷰티 랭킹 모멘텀 대시보드

올리브영·아마존 카테고리별 **랭킹 시계열**을 브랜드·제조사(ODM)·오너사 단위로 집계해
**투자 지표 관점**에서 보여주는 정적 웹 대시보드입니다. 수요 시그널(랭킹)과 상장사(오너/ODM)를
연결해 "어떤 종목이 K-뷰티 모멘텀을 타고 있나"를 한 화면에서 봅니다.

## 구성

| 화면 | 내용 |
|---|---|
| **메인** (`index.html`) | 올영 랭킹·ODM 점유·DART 재무 펀더멘털 통합 뷰 |
| **모멘텀 시그널** (`momentum.html`) | ⓪ 투자 종목 보드(상장, MoM 월간모멘텀順) · 비상장 IPO 워치 · 🇰🇷 아마존 K-뷰티 카테고리 레이더 |
| **제조사 심화** (`odm-deep.html`) | 상장 ODM 4사(콜마·코스맥스·코스메카·씨앤씨) 가동률·캐파·수익성 |

## 지표 정의

- **점수(7dMA)** = 랭킹 존재감. 상품 점수 `max(0, 101−순위)`의 브랜드·회사 합, 최근 7관측일 평균.
- **MoM** = 최근 30일 관측평균 ÷ 직전 30일 −1 (월간 모멘텀, 주지표).
- **WoW** = 주간 변화(참고용, 노이즈 큼).
- **아마존 K-뷰티 침투율** = 카테고리 Top50 중 한국 브랜드 SKU 비중.

빌드 도구·서버 불필요 — 순수 HTML/JS + Chart.js(CDN). GitHub Pages로 호스팅됩니다.

> 랭킹·재무는 공개 데이터 기반이며, 본 대시보드는 투자 참고용 정보이지 매수 권유가 아닙니다.
